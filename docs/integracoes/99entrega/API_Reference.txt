III. Cancel Order
1. Request Body
If the courier has picked up the package, order cancellation is not supported at this time.

Parameter Name	Type	Meaning	Required
external_order_id	string	External order ID (it’s recommended to use this parameter). If the value is empty, "order_id" will be used.	No
order_id	string	Unique order identifier	No
reason_id	string	Cancellation Reason ID
410013 // No courier accepted the order for a long time
410014 // Incorrect product information
410015 // Incorrect payment method selected
410016 // Incorrect delivery address
410017 // Incorrect recipient contact information
410018 // Delivery no longer needed
410019 // Courier has no change
410020 // Courier did not pick up the item
410021 // Unable to contact the courier	Yes
Note: Either external_order_id or order_id must be provided. If both are provided, only external_order_id will be used.

2. Response
Parameter Name	Type	Meaning	Required
errno	int32	Error code	Yes
errmsg	string	Error message	Yes
data	object	Response data	No

json
curl --location 'https://entrega.99app.com/entrega-openplatform/v2/order/cancel' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <Token>' \
--data '{
"order_id": "88000628207882",
"external_order_id":"",
"reason_id":"1"
}'
